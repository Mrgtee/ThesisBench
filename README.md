# ThesisBench

ThesisBench is a falsifiability layer for US stock AI trading agents.

It converts stock-market trading theses into structured event hypotheses, checks them against historical analogs, and returns an evidence-backed verdict before a paper trade is allowed.

## Hackathon Track

US Stocks AI Trading / Open Innovation.

## Safety

- Paper trading only.
- No real-money execution.
- API keys belong in `.env.local`, never in committed files.
